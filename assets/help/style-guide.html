🎨 راهنمای استفاده از فایل style.css

این فایل CSS مرکزی برای طراحی گرافیکی فرم‌ها، داشبوردها و صفحات پروژه استفاده می‌شود.

✅ فونت‌ها

  • Vazirmatn: برای متون فارسی
  • Inter: برای متون انگلیسی

✅ کلاس‌های دکمه

  • .btn-primary – آبی سازمانی
  • .btn-success – سبز تأیید
  • .btn-warning – نارنجی هشدار

مثال:

✅ پیام‌ها

✅ اطلاعات با موفقیت ثبت شد
⚠️ لطفاً همه فیلدها را پر کنید

✅ حالت تاریک

برای فعال‌سازی حالت تاریک، در تگ <body> بنویسید:

<body class="dark-mode">

✅ طراحی جدول

جدول‌ها دارای سایه ملایم، رنگ هدر، و واکنش‌گرایی در موبایل هستند.

📦 مسیر فایل

  • مسیر فایل: assets/css/style.css
  • قابل استفاده در: همه فرم‌ها و صفحات پروژه

📌 نکات توسعه آینده

  • افزودن کلاس‌های outline برای دکمه‌ها
  • افزودن کلاس‌های alert با آیکون
  • افزودن کلاس‌های فرم گروهی (form-group)

این فایل راهنما می‌تواند در مسیر assets/help/style-guide.html ذخیره شود و به‌عنوان مرجع طراحی استفاده گردد.
