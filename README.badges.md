## 📘 How to Use These Badges

- Copy Markdown badge code into `README.md`, `README.index.md`, or dashboard files  
- Update badge status manually or via GitHub Actions  
- Use consistent color coding:  
  - ✅ Green = Complete  
  - 🟡 Yellow = In Progress  
  - 🔵 Blue = Stable  
  - ⚪️ Grey = Planned
